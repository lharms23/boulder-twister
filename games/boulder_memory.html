Boulder Memory

So wird gespielt:

  • Der erste Spieler wählt einen Startgriff im Raster aus.
  • Der Kletterer muss die bisherige Sequenz klettern und am Ende einen **neuen Griff** hinzufügen.
  • Tippe den neuen Griff im Raster an – er wird automatisch nummeriert.
  • Die Wand füllt sich organisch. Wer die Reihenfolge vergisst oder den neuen Zug nicht schafft, fliegt raus!
